# Pynguin: Python for Penguin
![](./pynguin-logo.jpg)

### 一些不成熟的想法

本课程为问题导向型。每一单元围绕着一个具体问题展开，在试图解决具体问题的过程中学习必要的编程工具。

试图涵盖以下内容：

1. Unix入门；如何着手编写Python程序；Jupyter notebook入门。
2. Python基础语法
3. Numpy入门
4. Matplotlib入门
5. Pandas入门：处理社会科学数据
6. 网页爬虫
7. 贝叶斯统计？
